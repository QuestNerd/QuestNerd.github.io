Spin it before you buy it

Every model listing can support an interactive 3D preview.

Embed any .glb / .gltf file into a product page by dropping a <model-viewer> element into the HTML — visitors can rotate, zoom, and inspect the model in their browser, on mobile or desktop, with no plugins.

<model-viewer src="model.glb" alt="Model name" camera-controls auto-rotate></model-viewer>

Tip: include the <model-viewer> Web Component on the page and point src at any hosted .glb file.

# Site settings (Decap CMS singletons)

This directory holds singleton settings files written by the
**Site settings** collection in /admin/ (Decap CMS).

- `makerworld-print.json` — overrides for the four hour tiers and the
  intro paragraph on [`makerworld-prints.html`](../../makerworld-prints.html).
  When this file is absent, the defaults in
  [`assets/js/makerworld-config.js`](../../assets/js/makerworld-config.js)
  apply.

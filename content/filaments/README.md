# Add filament colors via /admin/

Each filament color is one JSON file in this directory. Use the
**Filament colors** collection in /admin/ (Decap CMS) to create them
through a form, or drop in JSON files by hand. See
[`assets/js/makerworld-config.js`](../../assets/js/makerworld-config.js)
for the starter list.

The on-site order form on `makerworld-prints.html` merges the entries
here on top of the defaults.
